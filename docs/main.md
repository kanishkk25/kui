# Introduction

KUI (Kanishk User Interface) is a lightweight terminal user interface library designed for the C programming language.

The purpose of KUI is to simplify the development of interactive command-line applications by providing higher-level abstractions over VT100 terminal functionality.

Instead of manually handling escape sequences, cursor positioning, colors, menus, user selections, checkboxes, and terminal rendering logic, developers can utilize KUI's API to create responsive text-based interfaces quickly and efficiently.

KUI is intended for:

<ul>
<li>Command-line tools</li>
<li>System administration utilities</li>
<li>Interactive terminal applications</li>
<li>Embedded Linux environments</li>
<li>Educational projects</li>
<li>Terminal dashboards</li>
<li>Configuration utilities</li>
</ul>

# Getting Started

## Prerequisites

Before using KUI, ensure that:

<ul>
<li>A C compiler is installed (GCC, Clang, MinGW, etc.)</li>
<li>Your terminal supports VT100 escape sequences</li>
<li>The KUI header and library files are accessible to the compiler</li>
</ul>

<hr>

## Installation

Place the following files in directories of your choice:

<ul>
<li><b>kui_vt100.h</b></li>
<li><b>libkui.a</b></li>
</ul>

Include the header file in your application:

<pre>
#include &lt;kui_vt100.h&gt;
</pre>

<hr>

## Compilation

Compile your application using:

<pre>
gcc your_source_code.c \
    -I path_to_header_directory \
    -L path_to_library_directory \
    -lkui \
    -o executable_name
</pre>

Example:

<pre>
gcc demo.c -I ./include -L ./lib -lkui -o demo
</pre>
